To work on this lab on your own machine, you will need to install Python and the Requests library.

Install Python

  • Install Python 3.4.2 or higher for your operating system.
  • Download the installer here: https://www.python.org/downloads/

Install the Requests Library

  • You can read more about the Requests Library here
  • To install Requests for Python 3 on Mac OS, use these command line commands: 
    • sudo curl -O https://raw.githubusercontent.com/pypa/pip/master/contrib/get-pip.py
sudo python3 get-pip.py
pip3 install requests
